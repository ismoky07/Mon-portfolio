Système Intelligent de Recherche dans des Documents (RAG)

Application de Retrieval-Augmented Generation (RAG) qui extrait le contenu de pages web et PDFs publics, indexe le contenu et permet des questions en langage naturel

RAG
Projet Personnel

Description

Application de Retrieval-Augmented Generation (RAG) qui extrait le contenu de pages web et PDFs publics via FireCrawl, indexe le contenu dans une base vectorielle, puis permet de poser des questions en langage naturel sur le contenu extrait avec des réponses contextualisées. Le système combine extraction intelligente, indexation sémantique et génération de réponses pour offrir une expérience de recherche documentaire avancée.

Pipeline RAG

Extraction de Contenu

Crawling et extraction de contenu web
  • Crawling et extraction de contenu web via FireCrawl
  • Parsing HTML via BeautifulSoup pour nettoyer et structurer les données extraites

Indexation Vectorielle

Chunking sémantique et recherche par similarité
  • Chunking sémantique des documents
  • Indexation vectorielle pour la recherche par similarité via l'agent Agno

Question-Réponse Contextuel

Chat en langage naturel avec réponses contextualisées
  • Interface de chat permettant de poser des questions en langage naturel
  • Réponses générées par GPT-4o basées sur le contexte extrait des documents

Architecture Technique

Extraction & Parsing

FireCrawl pour le crawling web intelligent et BeautifulSoup pour le parsing HTML et nettoyage du contenu.

Framework Agent

Agno pour l'orchestration de l'agent RAG avec gestion du contexte et des embeddings vectoriels.

Génération

OpenAI GPT-4o pour la génération de réponses contextualisées à partir des chunks les plus pertinents.

Technologies

FireCrawl Agno Framework OpenAI GPT-4o BeautifulSoup Streamlit Python

Points clés
