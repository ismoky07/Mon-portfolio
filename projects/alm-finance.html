Système Intelligent de Gestion Actif-Passif

Système intelligent d'IA pour la gestion Actif-Passif bancaire avec 6 agents spécialisés, conformité Basel III et dashboard interactif

TALENT PERFORMER
Juillet 2025 - Janvier 2026
AI Engineer

Description

Développement d'un système intelligent de gestion Actif-Passif avec 6 agents IA spécialisés pour automatiser l'évaluation des risques financiers, la gestion du capital, l'optimisation de la rentabilité et les opérations de trésorerie. Cette plateforme intelligente orchestre 6 agents IA spécialisés pour automatiser l'évaluation des risques financiers, la gestion du capital, l'optimisation de la rentabilité et les opérations de trésorerie. Le système intègre une architecture multi-LLM avec fallback automatique (xAI Grok-3, OpenAI GPT-4, Claude, Mistral, DeepSeek), une base vectorielle PgVector pour la recherche sémantique dans les documents réglementaires, et un dashboard interactif avec streaming temps réel des réponses.

Les 6 Agents Spécialisés

Agent 1 : Liquidity & Interest Rate Risk Assessor

Évaluation des risques de liquidité et de taux

Outils : calculate_liquidity_gaps() → calculate_eve_sensitivity() → calculate_nii_sensitivity()

  • Calcul du LCR (Liquidity Coverage Ratio) avec seuil Basel III ≥100%
  • Analyse des gaps de liquidité par bucket temporel (0-30, 30-90, 90-180+ jours)
  • Sensibilité EVE (Economic Value of Equity) aux chocs de taux ±200 bps
  • Projection NII (Net Interest Income) sur 12 mois glissants
LCR ≥100%

Agent 2 : Solvency & Capital Strategist

Adéquation du capital et financement de contingence

Outils : simulate_contingency_funding() → optimize_balance_sheet()

  • Suivi des ratios de capital : CET1 ≥8%, Tier 1 ≥10%, Total ≥12%
  • Simulation de financement d'urgence avec 3 sources (Banque Centrale, ventes actifs, lignes de crédit)
  • Détection des triggers de contingence (LCR <110%, retrait dépôts >10%)
  • Optimisation du bilan pour améliorer l'efficacité du capital

Architecture Technique

Multi-LLM avec Fallback

Support de 5 fournisseurs LLM avec basculement automatique en cas d'échec. xAI Grok-3 (principal), OpenAI GPT-4o, Anthropic Claude Sonnet, Mistral Large, DeepSeek Chat. Sélection dynamique du provider via le dashboard sans redéploiement.

Base Vectorielle & Knowledge

PostgreSQL + PgVector hébergé sur Neon.tech pour la recherche sémantique hybride. 6 fichiers markdown de knowledge base réglementaire (Basel III, IRRBB, LCR). Chunking sémantique (1000 tokens, seuil 0.5) avec embeddings SentenceTransformer ou OpenAI. Upload dynamique de documents (MD, JSON, CSV, Excel, PDF).

API REST & Dashboard

API Flask avec endpoints par agent et streaming SSE pour les réponses longues. Dashboard interactif Tailwind CSS avec inspection des outils, upload de fichiers, configuration des API keys et instructions personnalisées par agent. Déploiement Heroku-ready avec Gunicorn et Docker Compose pour PostgreSQL local.
