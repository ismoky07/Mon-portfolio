Agent Conversationnel Intelligent et Persistant sur WhatsApp

Bot WhatsApp avec Letta comme agent compagnon persistant avec mémoire à long terme

Chatbot
Projet Personnel

Description

Bot WhatsApp utilisant Letta (anciennement MemGPT) comme agent compagnon persistant avec mémoire à long terme. L'agent maintient le contexte des conversations passées, se souvient des préférences de l'utilisateur et supporte les messages textes et images pour une expérience conversationnelle naturelle et continue.

Fonctionnalités

Agent Persistant Letta

Mémoire et contexte continus
  • Letta (MemGPT) comme moteur d'agent
  • Mémoire persistante des conversations
  • Construction d'un profil utilisateur au fil du temps

Support Multimédia

Texte et images
  • Traitement des messages textes et images
  • Analyse contextuelle des images envoyées
  • Réponses multimodales

Mémoire Long Terme

Système de mémoire hiérarchique
  • Core memory, recall memory, archival memory
  • Contexte maintenu sur de longues périodes
  • Apprentissage continu des préférences

Architecture Technique

Backend

FastAPI avec Uvicorn comme serveur ASGI haute performance pour le traitement des webhooks WhatsApp.

Agent Letta

Letta Client pour la gestion de l'agent persistant avec mémoire structurée et gestion automatique du contexte.

Messagerie

WhatsApp Business API pour la réception et l'envoi de messages multimédia (texte, images).

Technologies

FastAPI WhatsApp Business API Letta (MemGPT) Uvicorn Python

Points clés

  • Mémoire persistante
  • Support texte et images
  • Profil utilisateur dynamique
  • Contexte long terme
  • Agent compagnon IA
  • Serveur ASGI performant
